# 17 - Version 2.1 du framework

## Objectif

La version 2.1 améliore la robustesse du projet.

## Améliorations

- centralisation des fonctions dans `lib/common.sh` ;
- meilleure journalisation ;
- rollback semi-automatique ;
- tests de syntaxe ;
- workflow GitHub Actions ;
- Makefile ;
- meilleure organisation.

## Commandes

```bash
make test
make install
make audit
make rollback
```
